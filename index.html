How to Get Files

Stock Report — Seller Central → Reports → Inventory Reports → Amazon-fulfilled Inventory Report → Request → Download (.txt)

Listings — Download your Google Sheets inventory as .xlsx (File → Download → Microsoft Excel). Upload the .xlsx file directly.

Data syncs to cloud — updates are visible to all team members.

Report History: The system keeps up to 3 report snapshots. Each new upload is compared against previous reports to detect stock changes, new/removed items, and anomalies.
